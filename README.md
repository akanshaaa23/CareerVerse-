# 🚀 Career Verse

Career Verse is an AI-powered career guidance platform designed to help students and professionals explore technology career paths, discover required skills, and create personalized learning roadmaps.

## ✨ Features

- AI-powered career recommendations
- Personalized learning roadmaps
- Career assessment quiz
- Salary insights and career comparisons
- Progress tracking system
- Achievement badges
- Responsive design
- Dark/Light mode support
- Search and filter functionality
- User-friendly interface

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript
- Local Storage
- Responsive Web Design
## 🚀 Getting Started
Open the project folder
cd career-verse
Run the project
Simply open index.html in your browser.

🎯 Purpose

Career Verse helps users make informed career decisions by providing structured learning paths, skill recommendations, and industry insights in one place.
